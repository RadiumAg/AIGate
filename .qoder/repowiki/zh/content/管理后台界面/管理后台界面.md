## 简介
本文件面向 AIGate 管理后台界面，提供从整体设计理念到具体组件实现的完整 UI/UX 设计文档。内容涵盖：

## 项目结构
AIGate 使用 Next.js App Router 架构，采用 RSC（React Server Components）与客户端组件混合模式，结合 TailwindCSS 与自定义主题变量实现"液态玻璃"风格的主题系统；ECharts 负责数据可视化；Radix UI 提供无障碍基础组件；trpc 进行前后端数据通信。

## 核心组件
- **模块化仪表板布局**：从单一组件重构为模块化架构，包括侧边栏头部、导航和页脚三个独立组件，提供更好的可维护性和扩展性
- **液态玻璃主题系统**：通过 CSS 变量与 Tailwind 扩展，统一管理背景、卡片、弹出层、主色与边框等视觉元素，并支持明暗两套配色
- **国际化导航系统**：导航菜单支持中英文双语切换，使用国际化配置文件实现动态语言切换
- **增强的用户菜单**：集成个人资料、邮箱密码设置和退出登录功能，提供完整的用户管理体验
- **主题切换功能**：在页脚组件中实现完整的主题切换，支持本地存储持久化和系统偏好检测
- **数据可视化**：基于 ECharts 实现用量趋势、模型分布与地区热力图，具备响应式与交互提示
- **响应式网格**：仪表板采用栅格布局，按屏幕尺寸自动调整列数与间距
- **trpc 客户端**：在客户端组件中发起查询与变更，配合加载态与错误态反馈
- **Radix UI 组合**：对话框、滚动区域、选择器等提供无障碍与可访问性基础
- **Tabs 组件**：基于 Radix UI 实现的选项卡组件，支持标签页切换、内容区域渲染和状态管理
- **Sonner 通知系统**：现代化的通知提示系统，支持多种通知类型（成功、错误、警告、信息、加载）和主题适配

## 性能考量
- **模块化布局性能**：侧边栏组件采用独立渲染，避免不必要的重渲染；导航项使用 useMemo 优化渲染性能
- **国际化性能**：导航文案通过 useTranslation hook 获取，支持语言切换时的性能优化
- **图表性能**：仅在数据可用且容器存在时初始化 ECharts；监听窗口 resize 并及时重绘；组件卸载时释放实例
- **加载态**：骨架屏与旋转指示器减少感知延迟；避免不必要的重复渲染
- **样式体积**：通过 Tailwind 自定义主题变量集中管理，减少重复样式声明
- **交互反馈**：过渡动画与悬停效果适度使用，避免影响滚动与交互流畅性
- **Tabs 组件性能**：使用 React.forwardRef 优化渲染性能；状态切换使用 CSS 变量而非内联样式
- **Sonner 通知性能**：轻量级通知系统，避免阻塞主线程；支持批量通知和自动清理
- **深色模式优化**：深色模式样式通过 CSS 变量和 Tailwind 修饰符实现，避免运行时样式计算开销
- **最近IP请求组件优化**：break-all 格式避免了复杂的文本截断逻辑，减少了计算开销
- **设置页面性能**：使用 trpc 缓存机制避免重复查询；表单验证在客户端执行减少服务器负载
- **设置路由性能**：文件操作使用同步方式，避免异步操作带来的复杂性；错误处理快速失败
